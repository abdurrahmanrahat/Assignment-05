The DOM means Document Object Model which is an application programming interface (API) for HTML. If we want to get a element from HTML page in js and want to change it, we will definitely use DOM. there are many ways to get a element from HTML page. Like get element by id or class or tag name as well as people use query selector & query selector all. We can interact with HTML element using DOM.

There are five ways to select a HTML element using DOM. Mention here
1) document.getElementById
2) document.getElementsByClassName
3) document.getElementTagName
4) document.querySelector
5) document.querySelectorAll

Event Bubbling is a concept in the DOM. It happens when an element receives an event, and that event bubbles up to its parent and ancestor elements in the DOM tree until it gets to the root element.

A callback is a function that is passed as an argument to other function. This method allows one function to call other function. When you want a function to execute only after another function has completed its execution, we use callback functions in JavaScript.
